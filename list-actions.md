# Действия со списками

Перебор 
```code
  list = [1,2]
  for item in list:
    print(item)

```
Типичные ошибки:
1. Забыли отступ
2. Поставили лишний отсуп
3. Поставили отступ не там, где надо
4. Поставили отступ после цикла и он применил нижестоящую операцию ко всем элементам
5. Забыли поставить двоеточие
